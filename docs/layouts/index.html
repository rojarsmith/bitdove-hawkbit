Update Management for IoT Device

Bitdove© is a domain independent update management server that for rolling out software updates to constrained edge devices as well as more powerful controllers and gateways connected to IP based networking infrastructure.

Fully-managed service based on Eclipse hawkBit™

With Bitdove IoT Rollouts, we provide you with a fully-managed cloud service based on Eclipse hawkBit™ and new features. This way, you can benefit from the strengths of this software without the need to take care of the infrastructure and maintenance yourself.

Interfaces

Bitdove offers a direct device integration via HTTP or a device management federation API which allows to connect devices with different protocol adapter. Users can make use of the graphical user interface and other service can interact with Bitdove through the RESTful management API.

Rollout

Bitdove supports an easy and flexible rollout management which allows you to update a large amount of devices in separated groups.

  • Cascading start of the deployment groups based on installation status of the previous group.
  • Emergency shutdown of the rollout in case a group exceeds the defined error threshold.
  • Rollout progress monitoring for the entire rollout and the individual groups.


Package Model


A software update does not always contain only a single file. The Bitdove meta model allows you to configure your files in virtual software and distribution packages.
